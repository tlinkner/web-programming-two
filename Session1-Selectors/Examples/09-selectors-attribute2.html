Attribute Selector Variations

selector[attribute] — matches elements that have the attribute

selector[attribute*="string"] — matches elements that have the string in the attribute value

selector[attribute^="string"] — matches any elements that have a value that begins with the string

selector[attribute$="string"] — matches any elements that have a value that ends with the string

selector[attribute~="string"] — matches any elements that have the string within a space separated list of values in the attribute

selector[attribute="string"] — matches any elements that have the exact string within the attribute
